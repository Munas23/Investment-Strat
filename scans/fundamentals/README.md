# Fundamental Screener - Pure Growth (Option B)

## Overview

The Fundamental Screener applies **Pure Growth** methodology based on Mark Minervini and William O'Neil's approaches. It screens liquid stocks (Tier 1-3 from liquidity screening) for aggressive growth characteristics: high earnings acceleration, strong revenue growth, and institutional backing.

**Configuration**: Option B - Pure Growth (Minervini/O'Neil style)
- Higher growth thresholds (25%+ EPS, 20%+ Revenue)
- Relaxed debt limits (50% vs 30%)
- No valuation filters (growth investors accept high multiples)

## Philosophy

**"Quality First, Timing Second"**

> We can't time our way out of a fundamentally poor business. The fundamental screen ensures we only trade institutional-quality growth companies with sustainable competitive advantages.

## Process Flow

```
1. Liquidity Screener → Identifies tradeable stocks (Tier 1-3)
                         ↓
2. Fundamental Screener → Scores quality (0-150 points, 60%+ to pass)
                         ↓
3. Technical Screener → Times entry (trend strength, breakouts)
                         ↓
4. Conviction Scoring → Position sizing (Levels 1-5)
                         ↓
5. Execution → Trade with confidence
```

## Scoring System (150 Points Total)

### Weighted Categories

```
Market Metrics:      20 pts (13.3%)
Profitability:       35 pts (23.3%)
Growth:              50 pts (33.3%) ← HIGHEST WEIGHT
Financial Strength:  25 pts (16.7%)
Institutional:       20 pts (13.3%)
                    ──────
Total:              150 pts (100%)
```

**Minimum Pass**: 90/150 points (60%)

## Detailed Scoring Breakdown

### 1. MARKET METRICS (20 points = 13.3%)

#### Market Cap (10 points)
```
Target Range:
├── USD: $300M - $50B
└── AUD: $500M - $50B (ASX stocks)

Scoring:
├── Within range: 10 pts ✅
├── Below minimum: Proportional (e.g., $150M = 5 pts)
└── Above maximum: 5 pts (mega-cap = slower growth)
```

**Why It Matters**: Optimal size for growth potential + institutional participation

#### Price (5 points)
```
Minimum:
├── USD: $15+
└── AUD: $5+

Scoring:
├── Above minimum: 5 pts ✅
└── Below minimum: Proportional
```

**Why It Matters**: Avoids penny stocks, indicates established company

#### Volume (5 points)
```
Minimum:
├── USD: 500,000 shares/day
└── AUD: 100,000 shares/day

Scoring:
├── Above minimum: 5 pts ✅
└── Below minimum: Proportional
```

**Why It Matters**: Confirms institutional interest and tradability

---

### 2. PROFITABILITY (35 points = 23.3%)

#### ROE - Return on Equity (20 points)
```
Scoring:
├── Excellent (25%+):  20 pts ✅✅✅
├── Good (20-24.9%):   15 pts ✅✅
├── Pass (15-19.9%):   10 pts ✅
├── Below 15%:         Proportional
└── Negative:          0 pts ❌
```

**Why It Matters**: Best measure of how efficiently company uses shareholder equity

#### Profit Margin (15 points)
```
Scoring:
├── Excellent (20%+):  15 pts ✅✅✅
├── Good (15-19.9%):   12 pts ✅✅
├── Pass (10-14.9%):   8 pts ✅
├── Below 10%:         Proportional
└── Negative:          0 pts ❌
```

**Why It Matters**: Shows pricing power and operating efficiency

---

### 3. GROWTH (50 points = 33.3%) ⭐ MOST IMPORTANT

#### Revenue Growth (25 points)
```
Scoring:
├── Excellent (30%+):  25 pts ✅✅✅
├── Good (25-29.9%):   20 pts ✅✅
├── Pass (20-24.9%):   15 pts ✅ ← RAISED from 15%
├── Below 20%:         Proportional
└── Negative:          0 pts ❌
```

**Why It Matters**: Top-line growth = expanding business, market share gains

**Minervini/O'Neil Standard**: 25%+ minimum (we allow 20%+ with lower scoring)

#### Earnings Growth (25 points)
```
Scoring:
├── Excellent (40%+):  25 pts ✅✅✅
├── Good (30-39.9%):   20 pts ✅✅ ← RAISED from 25%
├── Pass (25-29.9%):   15 pts ✅ ← RAISED from 18%
├── Below 25%:         Proportional
└── Negative:          0 pts ❌
```

**Why It Matters**: Bottom-line growth = sustainable profitability improvement

**Combined**: Revenue + Earnings = 50 points (33% of total score)

This is intentional - **growth drives stock price appreciation**

---

### 4. FINANCIAL STRENGTH (25 points = 16.7%)

#### Current Ratio (15 points)
```
Formula: Current Assets / Current Liabilities

Scoring:
├── Excellent (2.0+):   15 pts ✅✅✅
├── Good (1.8-1.99):    12 pts ✅✅
├── Pass (1.5-1.79):    8 pts ✅
├── Below 1.5:          Proportional
└── Below 1.0:          0 pts ❌ (liquidity crisis)
```

**Why It Matters**: Can company pay short-term obligations? Survival first.

#### Debt-to-Equity (10 points)
```
Formula: Total Debt / Shareholder Equity

Scoring (Lower = Better):
├── Excellent (≤0.20):  10 pts ✅✅✅ (20% debt or less) ← RELAXED from 0.10
├── Good (0.21-0.40):   8 pts ✅✅ (21-40% debt) ← RELAXED from 0.20
├── Pass (0.41-0.50):   5 pts ✅ (41-50% debt) ← RELAXED from 0.30
├── 0.51-1.0:           2 pts ⚠️
└── Over 1.0:           0 pts ❌ (more debt than equity)
```

**Why It Matters**: Low debt preferred, but growth companies can use leverage

**Change Rationale**: Pure growth investors (Minervini accepts <30%, Lynch <50%) tolerate moderate debt for growth. Relaxed from conservative 30% max to 50% max.

---

### 5. INSTITUTIONAL OWNERSHIP (20 points = 13.3%)

```
Optimal Range (50-70%):  20 pts ✅✅✅
Good Range (40-49% or 71-80%):  15 pts ✅✅
Too Low (<20%):          0 pts ❌ (no institutional interest)
Too High (>95%):         0 pts ❌ (no buyers left)
Other:                   5 pts ⚠️ (partial credit)
```

**Why It Matters**:
- **Too Low**: Institutions haven't discovered it yet (risky)
- **Optimal**: Strong institutional support, but room for more buying
- **Too High**: No new buyers, institutions may sell

**Sweet Spot**: 50-70% = Institutional validation + upside potential

## Usage

### Prerequisites

1. **Download symbols**:
   ```bash
   cd scans
   python download_the_csv.py
   ```

2. **Run liquidity screener** (required first):
   ```bash
   python "liquidity screener/liquidity_screener.py"
   ```
   This creates `liquidity_screen_*_TRADEABLE.csv` with Tier 1-3 stocks.

### Running the Fundamental Screener

```bash
cd scans
python fundamentals/fundamental_screener.py
```

**Process**:
1. Automatically finds latest download folder
2. Loads liquid stocks (Tier 1-3) from liquidity screening
3. Fetches fundamental data from Yahoo Finance for each stock
4. Scores on 10 weighted criteria (150 points total)
5. Assigns grade (A+, A, B+, B, C, D, F)
6. Saves 3 CSV files to download folder

**Runtime**: ~15-25 minutes for 1,000+ stocks (with API rate limiting)

## Key Insights

### Why This Scoring System Works

1. **Growth-Weighted** (33%): Stock prices follow earnings growth
2. **Quality-Filtered** (23%): Profitability ensures sustainability
3. **Safety-Checked** (17%): Financial strength prevents blowups
4. **Institutional Validated** (13%): Smart money confirmation
5. **Size-Appropriate** (13%): Optimal range for growth + liquidity

### What Separates Winners from Losers

**A+ Stocks (85%+) typically have**:
- Revenue growth: 30%+
- Earnings growth: 40%+
- ROE: 25%+
- Minimal debt (<10%)
- Strong institutional backing (50-70%)

**F Stocks (<40%) typically have**:
- Negative or minimal growth (<10%)
- Poor profitability (ROE <10% or negative)
- High debt (>100% of equity)
- Little institutional interest (<20%)

**The Difference**: Quality compounds, junk deteriorates
